The action space indices correspond to game moves as follows:

- Index 0: Add 1 to the shared number, valid if the number is prime.
- Index 1: Subtract 1 from the shared number, valid if the number is prime.
- Index 2: Divide the shared number by 2, valid if the number is even.
- Index 3: Multiply the shared number by 3 and add 1, valid if the number is odd.
