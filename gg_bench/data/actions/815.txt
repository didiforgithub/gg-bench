The action space indices map to the moves as follows:

- Index 0: Add 1 to the current number.
- Index 1: Add 2 to the current number.
- Index 2: Add 3 to the current number.
- Index 3: Add 4 to the current number.
- Index 4: Add 5 to the current number.
- Index 5: Multiply the current number by 2.
- Index 6: Multiply the current number by 3.
