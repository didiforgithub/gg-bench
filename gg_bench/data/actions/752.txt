The action space indices represent the eight possible moves for the Seeker in the game:

- Index 0: Move Up (-1 row, 0 columns)
- Index 1: Move Up-Right (-1 row, +1 column)
- Index 2: Move Right (0 rows, +1 column)
- Index 3: Move Down-Right (+1 row, +1 column)
- Index 4: Move Down (+1 row, 0 columns)
- Index 5: Move Down-Left (+1 row, -1 column)
- Index 6: Move Left (0 rows, -1 column)
- Index 7: Move Up-Left (-1 row, -1 column)
